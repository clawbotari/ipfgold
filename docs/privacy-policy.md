# Política de privacidad / Privacy Policy

## Español

**IPFGold** es una aplicación que muestra el precio del oro en tiempo real y datos históricos.

### Recopilación de datos
- **No recopilamos ningún dato personal.** La app no requiere registro, no solicita nombre, email, ubicación ni ningún otro dato personal.
- **No usamos publicidad.** No hay anuncios de terceros ni seguimiento para publicidad.
- **Los datos de precios** provienen de la API de Alpha Vantage (https://www.alphavantage.co). La app envía solicitudes HTTP para obtener cotizaciones de XAU/USD y tipos de cambio USD/EUR. Estas solicitudes no contienen información personal.

### Almacenamiento local
- La app almacena temporalmente los precios en una base de datos local (Room) para mejorar el rendimiento y reducir llamadas a la API. Los datos se eliminan automáticamente tras 7 días.

### Permisos
- La app no requiere permisos especiales (cámara, micrófono, contactos, etc.).
- Solo necesita conexión a Internet para actualizar los precios.

### Contacto
Si tienes preguntas sobre esta política de privacidad, puedes contactar con el desarrollador en:  
clawbotari@gmail.com

## English

**IPFGold** is an application that displays real-time gold prices and historical data.

### Data collection
- **We do not collect any personal data.** The app does not require registration, nor does it ask for name, email, location, or any other personal information.
- **We do not use advertising.** There are no third-party ads or tracking for advertising purposes.
- **Price data** comes from the Alpha Vantage API (https://www.alphavantage.co). The app sends HTTP requests to obtain XAU/USD quotes and USD/EUR exchange rates. These requests contain no personal information.

### Local storage
- The app temporarily stores prices in a local database (Room) to improve performance and reduce API calls. Data is automatically deleted after 7 days.

### Permissions
- The app does not require special permissions (camera, microphone, contacts, etc.).
- It only needs an internet connection to update prices.

### Contact
If you have questions about this privacy policy, you can contact the developer at:  
clawbotari@gmail.com
