# ipfgold – Android Gold Price Tracker

A real‑time gold price tracking Android application with historical charts, built with modern Kotlin and Jetpack Compose.

## Overview
- **Target users:** Individuals and families, not financial experts.
- **Supported currencies:** EUR and USD.
- **Visual style:** Financial / professional.
- **Technology stack:** Kotlin, Jetpack Compose, MVVM, Coroutines, Retrofit.

## Current Status
Project scaffold created. Requirements document in progress.

## Getting Started
Clone the repository and open in Android Studio (latest stable version).
